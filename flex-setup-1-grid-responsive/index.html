Grid on browsers, column on mobile

This demo has a media query that sets a column layout for mobile. The default is a flex grid. The flex container MAIN is set to be a row that wraps. Each SECTION item starts with a fixed width of 200px. You can customize these for each section.

Note that the images will scale up and down to the max width of the section, so you want to make sure your image resolution is as least as wide as your section width.
